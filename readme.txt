Easily manage testimonials and display them throughout your website.

== Description ==

This plugin adds a 'testimonial' post type to WordPress. Once you have created all your testimonials you can display a random one in any page or post simply by using the shortcode '[bamboo-testimonial]'. You can also display a list of all testimonials using the shortcode '[bamboo-testimonials]'.

Usage

For a single, random testimonial insert the bamboo-testimonial shortcode into your content as follows:

    [bamboo-testimonial]

For a list of all testimonial insert the bamboo-testimonials shortcode into your content as follows:

    [bamboo-testimonials]
